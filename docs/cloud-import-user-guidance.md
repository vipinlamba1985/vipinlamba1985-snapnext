# Cloud import user guidance

The Cloud Sync screen must clearly explain these limits before and during every import:

- Users may select up to **500 files in one import job**.
- SnapNext automatically processes that selection in **batches of 10 files**.
- Users do not need to manually create batches.
- Previously saved files are skipped safely.
- If a later batch fails, completed files remain saved.
- The progress panel should always show the current batch, completed files, saved files, skipped duplicates, and failed files.

Customer-facing copy should avoid technical terms such as API request, serverless timeout, queue worker, or provider limit.
