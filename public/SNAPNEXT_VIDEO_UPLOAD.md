# SnapNext hero video asset

This branch updates `app/page.js` to use a cinematic hero video at:

```txt
/public/snapnext-hero.mp4
```

To complete the live video test, upload the generated MP4 video to the repository at:

```txt
public/snapnext-hero.mp4
```

Optional poster image path:

```txt
public/snapnext-hero-poster.jpg
```

The landing page is already wired for autoplay, muted, looped playback with a pause/play control.

Note: the ChatGPT GitHub connector available in this session can update text files, but it cannot directly upload the binary MP4 asset from the chat into GitHub. The video file needs to be added manually through GitHub web upload, Git LFS, or local git.
